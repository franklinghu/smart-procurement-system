# 智能工业品采购系统 - UI 设计规范

## 一、设计目标

打造专业、高效、视觉惊艳的工业品采购平台，体现科技感与实用性。

## 二、目标用户

- 采购专员：日常采购操作，追求效率
- 部门经理：审批决策，关注数据清晰度
- 超级管理员：系统管理，追求简洁管理

## 三、风格定位

**主风格**：科技工业风 + 现代简约

**色调方案**：
- 主色：深蓝色 `#1E3A5F`（专业信任）
- 辅色：科技蓝 `#0066FF`（交互强调）
- 强调色：橙色 `#FF6B35`（重要操作）
- 背景：浅灰白 `#F8FAFC` / 深灰 `#1A1A2E`
- 成功：`#10B981`，警告：`#F59E0B`，错误：`#EF4444`

**字体**：
- 标题：思源黑体 / Noto Sans SC（Bold）
- 正文：思源黑体 / Noto Sans SC（Regular）
- 数据：JetBrains Mono（数字显示）

## 六、交互规范

- 页面加载：骨架屏 + 渐入动画
- 按钮点击：涟漪效果
- 卡片悬停：轻微上浮 + 阴影加深
- 列表切换：淡入淡出
- 表单验证：即时反馈 + 抖动提示
- 审批操作：二次确认弹窗

## 七、响应式断点

- 桌面：≥1200px（主要适配）
- 平板：768px - 1199px
- 移动端：< 768px（基础适配）
